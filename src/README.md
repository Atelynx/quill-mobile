# Carpeta `src`

Contiene el código fuente de la aplicación móvil.

- `api`: cliente HTTP para backend real.
- `components`: componentes reutilizables de interfaz.
- `config`: variables públicas de entorno.
- `constants`: valores estáticos de diseño.
- `hooks`: hooks compartidos.
- `mocks`: datos demo.
- `screens`: pantallas principales.
- `services`: repositorios, validaciones y realtime.
- `state`: sesión y preferencias.
- `styles`: estilos compartidos.
- `types`: contratos TypeScript.
- `utils`: funciones puras de formato y conversión.

La arquitectura separa navegación, UI, estado y acceso a datos para facilitar evolución hacia backend real.
