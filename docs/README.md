# Carpeta `docs`

Contiene documentación técnica complementaria de la app móvil.

- `backend-connection.md`: guía de conexión desde Android físico o emulador al backend NestJS local.
- `security-and-audit.md`: estado de persistencia segura, 401 y vulnerabilidades npm.
- `mobile-web-parity-audit.md`: auditoría de paridad funcional web/mobile para usuarios normales, excluyendo admin.
- `pre-apk-checklist.md`: criterios de sesión sin auto-login, safe area y checklist de validación antes de generar APK.

Esta documentación complementa el `README.md` raíz de `quill-mobile`.
